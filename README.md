AI-Powered Code Reviewer (Using React.js and Node.js Stack)

A full-stack AI-powered Code Reviewer web application built with Express.js, React and Node.js that allows developers to submit source code and get automated reviews, feedback, and analysis using AI. The app features syntax-highlighted code input using Prism.js for better readability and developer experience.

-> Features

* AI-based Code Analysis – Automatically analyzes submitted code to detect issues like syntax problems, logic errors, performance tips, and more. ( Uses Google Gemini-API )

* Syntax Highlighting with Prism.js – Uses Prism.js to highlight code in the editor and review display for a better visual experience.

* React frontend, Node.js/Express.js backend.

* Responsive UI – Intuitive UI for submitting code and viewing results (code editing with highlighted syntax).

* Multi-language Support – Works with multiple programming languages (e.g., JavaScript, Python, etc.).
